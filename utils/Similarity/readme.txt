1-ый параметр: путь до дирректории с zip файлами либрусека.
2-ой параметр: название файла, куда будут сохраняться дубли, найденные MinHash
3-ый параметр: название файла, куда будут сохраняться дубли, найденные старым алгоритмом, на основе ключа "Имя автора + фамилия автора + название книги"

Пример:
java -jar similarity.jar D:\data\test\zip\ D:\data\test\minhash_similar_docs.txt D:\data\test\similarby_key_docs.txt
