1-ый параметр: путь до дирректории с zip-архивами либрусека
2-ой параметр: путь до дирректории, куда будет сохраняться индекс
3-ий параметр: путь до файла, в котором перечислены названия файлов-дубликатов, которые не нужно индексировать

Пример:
java -jar indexer.jar D:\data\test\zip\ D:\data\test\index\ D:\data\test\same_docs.txt
