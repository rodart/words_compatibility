1-ый параметр: путь до дирректории с индексом
2-ой параметр: название файла, куда будут сохраняться предложения, найденные по запросу
3-ый параметр: сам запрос
4-ый параметр: количетсво предложений, которые нужны по данному запросу (-1 - все возможные)

Пример:
java -jar searcher.jar D:\data\test\index\ D:\data\test\query_result.txt палка -1
